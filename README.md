## 內容

主要是Linux開發的內容，當然任何和Linux有關的東西也有可能出現。

參考的書有很多，大概這三本：

- ***The Linux programming interface a Linux and UNIX system programming handbook by Michael Kerrisk***
- ***The Linux Command Line A Complete Introduction by William E. Shotts***
- ***APUE***
